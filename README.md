# 🏙️ SimCity 2000 모작

클래식 도시 건설 시뮬레이션 **SimCity 2000**을 모작한 프로젝트입니다. 플레이어는 도로를 건설하고, 전력을 공급하며, 다양한 건물을 배치하여 도시를 성장시킬 수 있습니다.
